# 📘 Ingeniería II – Tecnicatura Superior en Desarrollo de Software

Repositorio correspondiente a la materia **Ingeniería II** de 2° año de la *Tecnicatura Superior en Desarrollo de Software*.

## 🧠 Descripción

Este repositorio contiene trabajos prácticos, ejercicios y proyectos desarrollados a lo largo de la cursada de la materia. El enfoque principal está en el diseño y desarrollo de aplicaciones web utilizando tecnologías fundamentales del desarrollo frontend.

## 🚀 Tecnologías utilizadas

A lo largo del curso trabajaremos con:

* **HTML5** → Estructura y contenido de las páginas web
* **CSS3** → Estilos, diseño y responsividad
* **JavaScript** → Interactividad y lógica del lado del cliente
* **Bootstrap** → Framework para diseño responsive y componentes UI

## 📂 Estructura del repositorio

El repositorio se organiza de la siguiente manera:

```
/proyectos
  /proyecto-1
  /proyecto-2

/practicas
  /html
  /css
  /javascript

/assets
  /img
  /css
  /js

index.html
README.md
```

> La estructura puede ir evolucionando a medida que avance la cursada.

## 🎯 Objetivos de la materia

* Comprender los principios de desarrollo web
* Aplicar buenas prácticas de diseño y programación
* Desarrollar interfaces web modernas y responsivas
* Integrar diferentes tecnologías frontend
* Trabajar de forma colaborativa utilizando control de versiones

## 🛠️ Cómo usar este repositorio

1. Clonar el repositorio:

   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```

2. Abrir el proyecto en tu editor de código preferido (por ejemplo, VS Code)

3. Ejecutar los archivos `.html` en tu navegador

## 📌 Requisitos

* Navegador web actualizado (Chrome, Firefox, Edge, etc.)
* Editor de código (recomendado: Visual Studio Code)
* Conocimientos básicos de programación
